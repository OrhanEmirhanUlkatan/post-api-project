

# Project Title

Post Api Porject

## Run Locally

Clone the project

```bash
  git clone https://github.com/OrhanEmirhanUlkatan/post-api-project.git
```

Go to the project directory

```bash
  cd post-api-project
```

Install dependencies

```bash
  npm install || yarn
```

Start the server

```bash
  npm start || yarn start
```
